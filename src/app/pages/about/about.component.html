Why QIzz?

QIzz stands for Quiz Inteligence. It is build for creating quizes with GPTs suports.

How it works?

  1. Go home
  2. Create a quiz. You can generate prompt for Chat-GPT or use quiz builder.
  3. Paste quiz template data (if you have)
  4. Load and check quiz
  5. Save quiz and share with others

Why URL is so long?

URL is long because it contains all quiz data.
You can share it with others and they will see the same quiz as you.
You can also share pure compressed data and load it in Load Quiz page.

This app is front-only
